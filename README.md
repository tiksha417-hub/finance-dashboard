# Personal Finance Dashboard
A real-time financial tracking application built with React, Tailwind CSS v4, and Recharts.

# Live Demo
[cool-souffle-b40f06.netlify.app]

# Features
- Admin/Viewer Roles: Toggle between permissions.
- Visual Insights: Area charts for balance trends.
- Search & Filter: Easily find transactions by category or description.
- Responsive Design: Fully functional on mobile and desktop.

# Tech Stack
- Frontend: React (Vite)
- Styling: Tailwind CSS
- Charts: Recharts
- Icons: Lucide-React
